# Frequently Asked Questions

## General

### What is Farm?

Farm is an open-source developer portal platform. It provides a centralized hub for managing software components, technical documentation, and user authentication.

### Why should my team use Farm?

Farm helps teams improve developer experience by providing a single source of truth for their software ecosystem. It reduces cognitive load, simplifies onboarding, and makes it easier to discover and understand existing services.

### Is Farm free to use?

Yes! Farm is open-source software licensed under the AGPL-3.0 license.

## Technical

### Which technologies does Farm use?

Farm's backend is built with NestJS and TypeScript. The documentation is managed with MkDocs and the Material for MkDocs theme.
